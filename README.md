# 🎵 SoundWave — Музыкальный портал

Одностраничный музыкальный сайт с подборками треков, аудиоплеером и интерактивными элементами. Итоговый проект по веб-разработке.

![Скриншот сайта](screenshot.png)

## 🌐 Живая версия

[Открыть сайт](https://username.github.io/soundwave)

## ✨ Функциональность

- 🎧 **Плеер с треками** — встроенный HTML5-аудиоплеер в каждой карточке
- 📂 **Динамическая загрузка** — кнопка «Показать еще» добавляет новые карточки
- 🌓 **Тёмная тема** — переключение светлой/тёмной темы с сохранением выбора
- 📝 **Форма обратной связи** — валидация полей, вывод данных в консоль
- 📱 **Адаптивный дизайн** — корректное отображение на мобильных устройствах
- 🎨 **Градиентный фон** — меняется в зависимости от темы
